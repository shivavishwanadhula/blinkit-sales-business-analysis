# 🛒 Blinkit Sales & Profitability Analysis

## Project Overview
This project analyzes Blinkit sales data to uncover trends in revenue, profitability, discounts, delivery performance, and customer purchasing behavior.

## Objectives
- Analyze revenue trends
- Identify profitable categories
- Study discount effectiveness
- Evaluate delivery performance
- Generate business recommendations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Top categories generate most revenue.
- Excessive discounts reduce profitability.
- Some cities outperform others in order volume.
- Delivery performance impacts customer experience.

## Recommendations
- Optimize discount strategies.
- Improve inventory planning.
- Focus marketing on high-profit categories.
- Improve delivery efficiency.
